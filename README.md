# Communion
Ritual binaural experience. Available for Linux, Windows, and macOS.

## Downloads
- **Linux:** `Communion_Linux.zip`
- **Windows:** `Communion_Windows.zip`
- **Mac:** `Communion_Mac.zip`

## How to Run

### Linux
1. Unzip `Communion_Linux.zip`
2. Open a terminal in the extracted folder
3. Run: `chmod +x Communion.v2.x86_64 && ./Communion.v2.x86_64`

### Windows
1. Unzip `Communion_Windows.zip`
2. Double-click `Communion.v2.exe`

### macOS
1. Unzip `Communion_Mac.zip`
2. Double-click `Communion.v2.app`
3. If blocked by Gatekeeper: right-click → Open

## Controls
- Hold N to engage the system
- Release to let everything fade
- Press Esc to exit
